Continuidad psicoemocional estructurada entre atenciones

SINAC-SM es un sistema orientado a fortalecer seguimiento, visibilidad y articulación institucional entre atenciones, abordando un vacío frecuente en contextos universitarios y organizacionales: la fragmentación del acompañamiento, la baja trazabilidad entre contactos y la dificultad para sostener continuidad sin convertir el sistema en atención clínica directa.

Ver MVP funcional Explorar propuesta
Importante: SINAC-SM no realiza diagnóstico, no entrega tratamiento psicológico o psiquiátrico, no reemplaza profesionales y no sustituye dispositivos de urgencia.

Propósito del sistema

Estructurar seguimiento entre atenciones, hacer visibles señales relevantes y facilitar continuidad institucional.

Problema institucional

El problema no es solo el acceso. También es la continuidad.

En muchos contextos institucionales, el acompañamiento psicoemocional no falla únicamente por falta de atención, sino por la pérdida de continuidad entre contactos, la escasa visibilidad longitudinal y la dificultad para sostener seguimiento estructurado entre una atención y otra.

Fragmentación del seguimiento

La continuidad suele depender de contactos aislados y no de una estructura que permita observar proceso, evolución y trayecto entre atenciones.

Baja visibilidad entre contactos

Lo que ocurre entre sesiones o espacios de apoyo rara vez queda organizado de forma útil para lectura longitudinal.

Derivación reactiva

Sin continuidad estructurada, muchas rutas de apoyo se activan tarde o con escasa trazabilidad institucional.

Propuesta del sistema

Qué propone SINAC-SM

SINAC-SM propone un sistema de continuidad psicoemocional estructurada que complementa la atención existente y organiza seguimiento entre atenciones. Su foco no está en intervenir clínicamente, sino en aportar estructura, continuidad y mejor visibilidad para apoyar marcos institucionales definidos.

Componentes principales

Registro estructurado

Permite dejar trazas breves de seguimiento entre contactos profesionales.

Continuidad longitudinal

Ordena información para dar mayor visibilidad al proceso entre atenciones.

Señales relevantes

Ayuda a identificar patrones simples de seguimiento sin automatizar decisión clínica.

Apoyo institucional

Facilita continuidad y articulación según criterios definidos por la propia institución.

Lógica general

1. Registro

El usuario deja información breve entre atenciones.

2. Organización

El sistema ordena esa información para seguimiento y lectura longitudinal.

3. Visibilidad

Se presentan señales o patrones básicos para apoyar continuidad.

4. Articulación

La institución define criterios, responsables y rutas de uso.

Límites del modelo

Lo que este sistema no es

Esta delimitación es esencial para mantener claridad estratégica, ética e institucional.

No realiza diagnóstico clínico.
No entrega tratamiento psicológico o psiquiátrico.
No reemplaza atención profesional.
No sustituye dispositivos de urgencia ni canales de emergencia.

Solicitud inicial

Gobernanza y responsabilidades

Un sistema con roles claramente diferenciados

El valor de SINAC-SM depende de una separación clara entre lo que hace el sistema, lo que define la institución y lo que sigue siendo responsabilidad de profesionales.

Sistema

Estructura seguimiento, organiza información, visibiliza señales simples y facilita continuidad no clínica.

Institución

Define alcances, criterios, protocolos, responsables, supervisión y rutas de uso.

Profesionales

Mantienen criterio clínico, evaluación, decisión e intervención. No son reemplazados por el sistema.
